## 一、主要情感分类理论

### 1.1 Ekman 基本情绪理论（1972）

Paul Ekman 通过跨文化面部表情研究，提出 **六种基本情绪**（后扩展至七种）：

| 基本情绪 | 英文 | 核心功能 |
|---------|------|---------|
| 快乐 | Happiness | 促进社会联结与合作 |
| 悲伤 | Sadness | 引发他人同情、促进反思 |
| 恐惧 | Fear | 威胁探测与逃避反应 |
| 愤怒 | Anger | 清除障碍、维护边界 |
| 厌恶 | Disgust | 拒绝有害物质/行为 |
| 惊讶 | Surprise | 注意力重定向 |
| 蔑视 | Contempt | 社会等级信号（后期增加） |

Ekman 的核心论据是：这些情绪的面部表情在包括巴布亚新几内亚等未接触现代文明的部落中仍可被识别，说明其具有**生物先天性**而非纯粹文化习得。

### 1.2 Plutchik 情绪轮模型（1980）

Robert Plutchik 从进化心理学出发，提出 **八种原始情绪**，两两对立：

```
       恐惧 ←→ 愤怒
       快乐 ←→ 悲伤
       信任 ←→ 厌恶
       惊讶 ←→ 期待
```

关键创新：

- **强度梯度**：每种情绪有不同强度（如 恼怒 → 愤怒 → 暴怒）
- **混合情绪**：两种基本情绪组合产生复合情绪（如 快乐 + 信任 = 爱；恐惧 + 惊讶 = 敬畏）
- 以色轮形式可视化，相邻情绪容易混合，对立情绪互相抑制

### 1.3 Russell 环形模型（Circumplex Model, 1980）

James Russell 提出维度取向的分类，不设离散类别，而用**两个连续维度**定位情感：

- **效价（Valence）**：愉悦 ←→ 不愉悦
- **唤醒度（Arousal）**：高激活 ←→ 低激活

示例：

| 效价 | 唤醒度 | 情绪状态 |
|------|-------|---------|
| 高效价 | 高唤醒 | 兴奋 |
| 低效价 | 高唤醒 | 焦虑 |
| 高效价 | 低唤醒 | 平静满足 |
| 低效价 | 低唤醒 | 抑郁消沉 |

此模型在情感计算（Affective Computing）领域应用广泛。

### 1.4 其他重要分类

- **William James**（1884）：将情绪分为"粗糙情绪"（如恐惧、愤怒，伴随强烈生理反应）和"精细情绪"（如审美感受、道德情感）
- **中国传统心理学**对应：七情说——喜、怒、忧、思、悲、恐、惊（《黄帝内经》）

## 二、人对情感的感知理论

这是心理学中争论最激烈的问题之一："我们如何知道自己在经历某种情绪？"

### 2.1 James-Lange 理论（1884/1885）

**核心主张**：生理变化在先，情感体验在后。

> "我们不是因为悲伤而哭泣，而是因为哭泣而悲伤。" —— William James

机制链条：

```
刺激事件 → 身体生理反应（心跳加速、肌肉紧张等） → 大脑感知到身体变化 → 产生情绪体验
```

每种情绪对应独特的生理反应模式，大脑通过识别不同的身体信号来区分不同情绪。

### 2.2 Cannon-Bard 理论（1927）

Walter Cannon 对 James-Lange 的批判。

**核心主张**：生理反应和情感体验**同时平行发生**，而非因果关系。

论据：
- 内脏反应太慢（需 1-2 秒），而情绪体验几乎瞬时
- 不同情绪的生理反应非常相似（恐惧和愤怒都心跳加速），无法仅靠生理信号区分
- 切断实验动物的内脏神经后，动物仍表现出情绪行为

机制：丘脑同时向大脑皮层（产生主观体验）和身体（产生生理反应）发送信号。

### 2.3 Schachter-Singer 二因素理论（1962）

**核心主张**：情绪 = 生理唤醒 + 认知标签

这是一个折中方案：
- 生理唤醒提供情绪的**强度**
- 认知评估（对环境的解读）决定情绪的**类型**

经典实验（肾上腺素实验）：注射肾上腺素的被试在快乐环境中报告快乐，在愤怒环境中报告愤怒——同样的生理唤醒被认知赋予了不同标签。

### 2.4 Lazarus 认知评价理论（1991）

Richard Lazarus 强调**认知评价（appraisal）是情绪产生的必要条件**：

- **初级评价**：这个事件与我有关吗？是威胁还是机遇？
- **次级评价**：我有资源应对吗？
- 两次评价的结果决定产生何种情绪

同一事件（如被裁员），不同的评价方式产生完全不同的情绪反应（恐惧、愤怒、或解脱）。

### 2.5 Lisa Feldman Barrett 构建情绪理论（2017）

较新但有影响力的理论。

**核心主张**：情绪不是被"触发"的，而是大脑主动**构建**的。

大脑基于过往经验、文化概念和当前身体状态的预测来构建情绪。不存在固定的"情绪指纹"——恐惧在不同场景、不同人身上的神经和生理模式各不相同。

### 2.6 各理论对比

| 理论 | 年代 | 核心观点 | 情绪产生顺序 |
|------|------|---------|------------|
| James-Lange | 1884 | 身体反应决定情绪 | 刺激 → 生理 → 情绪 |
| Cannon-Bard | 1927 | 生理与情绪同时发生 | 刺激 → 生理 + 情绪（平行） |
| Schachter-Singer | 1962 | 生理唤醒 + 认知标签 | 刺激 → 唤醒 → 认知解释 → 情绪 |
| Lazarus | 1991 | 认知评价是前提 | 刺激 → 认知评价 → 情绪 + 生理 |
| Barrett | 2017 | 大脑主动构建 | 预测 + 身体信号 + 概念 → 情绪 |

## 三、情感对内分泌系统的影响

情绪与内分泌系统形成双向调节回路：情绪触发激素分泌，激素水平反过来影响情绪状态。

### 3.1 HPA 轴（下丘脑-垂体-肾上腺轴）—— 压力/恐惧/焦虑

```
负面情绪（恐惧、焦虑、愤怒）
       ↓
下丘脑释放 CRH（促肾上腺皮质激素释放激素）
       ↓
垂体前叶释放 ACTH（促肾上腺皮质激素）
       ↓
肾上腺皮质释放 皮质醇（Cortisol）
```

**皮质醇**的作用：

- **短期**：动员能量、增强警觉、抑制非紧急功能（消化、免疫、生殖）
- **长期过高**：免疫抑制、海马体萎缩（损害记忆）、腹部脂肪堆积、胰岛素抵抗
- 长期慢性压力 → 皮质醇昼夜节律紊乱 → 抑郁症的生理基础之一

### 3.2 SAM 轴（交感神经-肾上腺髓质轴）—— 急性应激

```
突发恐惧/愤怒
       ↓
交感神经激活
       ↓
肾上腺髓质释放 肾上腺素（Epinephrine）+ 去甲肾上腺素（Norepinephrine）
```

效果（战斗或逃跑反应 Fight-or-Flight）：

- 心率加速、血压升高、瞳孔扩大
- 血糖升高、肌肉供血增加
- 消化功能抑制
- 反应在数秒内启动，数分钟内达峰

### 3.3 正面情绪的内分泌对应

| 情绪/状态 | 关键激素/神经递质 | 机制 |
|----------|---------------|------|
| 快乐/愉悦 | **多巴胺**（Dopamine） | 中脑边缘奖赏通路激活，产生愉悦感和动机 |
| 社会联结/信任/爱 | **催产素**（Oxytocin） | 身体接触、哺乳、社交互动时释放；降低皮质醇 |
| 平静/满足 | **血清素**（Serotonin） | 调节情绪基线、睡眠、食欲；低水平与抑郁关联 |
| 剧烈运动后的欣快 | **内啡肽**（Endorphins） | 内源性阿片类物质，镇痛和欣快感 |
| 亲密关系/依恋 | **加压素**（Vasopressin） | 与催产素协同，调节配对绑定和领地行为 |

### 3.4 情绪-内分泌的病理循环

- **抑郁症**：皮质醇持续升高 → 海马体体积缩小 → HPA 轴负反馈受损 → 皮质醇进一步升高（恶性循环）；血清素、多巴胺、去甲肾上腺素失衡
- **PTSD**：杏仁核过度激活 → 持续的恐惧条件化 → 肾上腺素系统过度敏感化
- **慢性焦虑**：皮质醇节律异常 → 免疫功能下调 → 躯体化症状

## 四、情感对决策的影响

### 4.1 Damasio 躯体标记假说（Somatic Marker Hypothesis, 1994）

Antonio Damasio 通过研究前额叶腹内侧（vmPFC）损伤的患者提出了最有影响力的理论。

**核心发现**：患者 Elliot 在 vmPFC 损伤后，智力测试正常，但完全无法做出日常决策（选餐厅、安排日程可能耗时数小时）。

**机制**：

1. 过往经验在身体中留下情绪"标记"（躯体标记）
2. 面临决策时，大脑自动重新激活与备选方案相关的身体感受
3. 这些"直觉"（gut feeling）帮助快速筛选选项，不需要逐一理性分析
4. 没有情绪参与的"纯理性"决策反而效率极低

**Iowa 赌博任务**证实：正常人在意识到哪组牌好之前，皮肤电反应已经对坏牌组产生回避信号——情绪系统比意识更早"知道"答案。

### 4.2 Kahneman 双系统理论中的情绪角色

Daniel Kahneman（2002 年诺贝尔经济学奖）的框架：

- **系统 1**（快速、直觉、自动）：大量依赖情绪启发式（affect heuristic）
- **系统 2**（缓慢、审慎、分析）：可以覆盖系统 1，但需要认知努力

**情感启发式（Affect Heuristic）**：人们用"我对此感觉如何"来替代"对此我怎么想"——对风险和收益的判断高度受情绪状态影响。

### 4.3 具体情绪对决策的不同影响

| 情绪 | 对决策的影响 | 机制 |
|------|-----------|------|
| **恐惧** | 风险回避增强、过度保守 | 高估威胁概率、降低风险承受度 |
| **愤怒** | 风险偏好增强、过度自信 | 增加确定性感知、降低对风险的敏感度 |
| **悲伤** | 倾向改变现状、愿意付更高价格 | 触发"改变处境"的动机 |
| **快乐** | 依赖启发式、减少深入分析 | "一切顺利"的信号减少了深思的动机 |
| **焦虑** | 决策瘫痪或冲动决策（两极化） | 不确定性放大、工作记忆容量被占用 |
| **厌恶** | 拒绝/丢弃倾向增强 | 泛化排斥反应至不相关领域 |

### 4.4 情绪对决策的关键效应

#### 框架效应（Framing Effect）

同一信息以"收益"或"损失"框架呈现，触发不同情绪，导致截然不同的选择。"手术存活率 90%"和"手术死亡率 10%"是同一事实，但后者引发恐惧，显著降低患者同意手术的概率。

#### 损失厌恶（Loss Aversion）

失去 100 元的痛苦约为获得 100 元快乐的 **2-2.5 倍**。这不是理性计算的结果，而是杏仁核对损失的情绪反应天然强于对收益的反应。

#### 即时情绪 vs. 预期情绪

- **即时情绪**（决策时的当下感受）直接扭曲判断
- **预期情绪**（想象未来会有的感受）是决策的重要输入
- 人们系统性地高估未来情绪的强度和持续时间（**影响偏差，Impact Bias**）

#### 热-冷共情差距（Hot-Cold Empathy Gap）

在情绪冷静时无法准确预测自己在情绪激动时会做什么决定，反之亦然。这解释了为什么"饱时不决断饥时事"具有心理学基础。

## 五、总结

传统心理学对情感的理解可以归纳为四个核心命题：

1. **情绪具有生物普遍性**，但其表达和体验受文化和认知的深度塑造
2. **情绪感知不是被动接收**，而是身体信号、认知解释和社会语境共同建构的结果
3. **情绪与内分泌系统形成双向因果环路**，短期适应性的应激反应在慢性化后转为病理
4. **情绪是决策的必要组件而非干扰**——纯理性决策是一个神话；但特定情绪状态会系统性地扭曲判断
